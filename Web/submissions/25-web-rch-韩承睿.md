### Linux基础指令（简洁版）
#### 一、目录与文件
pwd 查看当前路径
ls 列出文件
cd 切换目录
mkdir 创建文件夹
touch 创建空文件
rm 删除文件
rm -r 删除文件夹
cp 复制
mv 移动 / 重命名
二、查看文件
cat 查看文件内容
head 看前几行
tail 看后几行
less 分页查看
三、系统与进程
ps 查看进程
top 实时看资源占用
kill 结束进程
df -h 看磁盘空间
free -h 看内存
uname -a 看系统信息
四、权限与用户
chmod 修改权限
sudo 以管理员执行
su 切换用户
passwd 修改密码
五、网络
ping 测试网络
ip addr 看 IP
netstat 看端口
六、搜索与查找
grep 查找内容
find 查找文件
七、压缩解压
tar -zcvf 压缩
tar -zxvf 解压
