![562ecae58ddf21a07f139e4025f23c5c.png](https://i.mji.rip/2026/04/12/562ecae58ddf21a07f139e4025f23c5c.png)

提取出buffer的数据（2dup(0) 代表两个重复的0），先大致清洗一下，然后用python脚本实现去除空格和末尾的h，再用16进制解释成整数。接着对数据依次反向执行上面的操作并取模（ASCII的合法范围最大是0xff），解释成字符，最后拼接在一起。

```python
raw = """47h, 95h, 34h, 48h, 0A4h, 1Ch, 35h, 88h, 64h, 16h, 88h,7, 14h, 6Ah, 39h, 12h, 0A2h, 0Ah, 37h, 5Ch, 7, 5Ah,56h, 60h, 12h, 76h, 25h, 12h, 8Eh, 28h, 0 ,0"""

res = [int(s.strip().rstrip("h"), 16) for s in data]
for i in range(len(res)):
    if i % 3 == 0:
        res[i] += 31
    elif i % 3 == 1:
        res[i] -= 41
    else:
        res[i] ^= 0x55
    res[i] %= 0xFF
    res[i] = chr(res[i])

print("".join(res))
```
